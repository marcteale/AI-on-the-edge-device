Welcome to AI-on-the-edge Device Setup

This is AI-on-the-edge's initial setup procedure. The basic setup consists of seven steps, after which your device will reboot into the regular web interface.
Note: All initial configuration settings are also accessible using the regular web interface. See the documentation for additional information.

You can navigate during setup with the "Next Step" and "Previous Step" buttons.
The "Abort Setup" button will skip setup and you will be sent to the abort screen.
To restart, use the "Restart Setup" button.

Setup procedure overview:

  1. Adjust focus and check for LED reflections.
    Ensure your camera lens is properly focused and the LED does not create reflections that interfere with readings.

  2. Create the reference image.
    This is the base for position referencing and the identification of digits and counters.

  3. Define two unique alignment markers.
    Alignment markers are used to orient images before further processinig.

  4. Define regions of interest (ROI's) for digits.
    These will be used to digitize numerical parts of your meter.
    (Can be skipped if your meter does not have numerical indicators.)

  5. Define ROI's for analog counters.
    These will be used to digitize analog parts (dials) of your meter.
    (Can be skipped if your meter does not have analog counters.)

  6. Additional configuration.
    Any additional configuration can be done here. This can also be performed later with regular web interface, e.g. setting up publishing services like MQTT.

  7. Setup Completion. End/Abort setup mode.
    Setup must be properly terminated on this page or you will sent back to setup after rebooting.

Please be patient between steps. Devices can take quite some time to load all necessary information!

If you need support, have a look to the documentation or discussion pages.

Have fun with your AI-on-the-edge device!
